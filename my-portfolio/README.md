## ğŸ¨ Customization

### Update Your Information

1. **Personal Details**: Edit `components/HeroSection.tsx`, `AboutSection.tsx`, and `ContactSection.tsx`
2. **Projects**: Modify the `projects` array in `components/ProjectsSection.tsx`
3. **Skills**: Update the `skills` object in `components/SkillsSection.tsx`
4. **Social Links**: Change links in `components/ContactSection.tsx` and `Footer.tsx`
5. **Metadata**: Update `app/layout.tsx` for SEO

### Color Scheme

The portfolio uses a purple-pink-orange gradient theme. To change colors:
- Edit Tailwind classes in components (from-purple-400, via-pink-500, etc.)
- Modify `app/globals.css` for global color changes
