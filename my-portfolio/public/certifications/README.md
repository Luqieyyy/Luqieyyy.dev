## Recommended Specs:
- Format: PNG dengan transparent background (best!)
- Size: 200x200px to 512x512px (square)
- File size: Under 100KB
- High resolution untuk clarity

- Guna official brand logos
- Pastikan background transparent
- Square/circular logos works best
