# Education Logos

Letak logo sekolah/universiti kat sini.

## Logo Yang Diperlukan:

1. **uthm-logo.png** - Logo Universiti Tun Hussein Onn Malaysia
2. **matrik-logo.png** - Logo Matriculation College Melaka

## Recommended Specs:
- Format: PNG dengan transparent background (preferred)
- Size: 500x500px atau square ratio
- File size: Under 200KB
- High resolution untuk clarity

## Tips:
- Guna official logo dari website universiti
- Pastikan background transparent untuk best look
- Square/circular logos works best
- PNG format untuk maintain quality

## Where to Download:
- UTHM Logo: https://www.uthm.edu.my atau search "UTHM logo PNG"
- Matrik Melaka Logo: Search "Kolej Matrikulasi Melaka logo"
