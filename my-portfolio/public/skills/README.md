# Skills Icons/Images

Letak icon atau images untuk skills section kat sini.

## Images Yang Diperlukan:

1. **frontend-icon.png** - Icon untuk Frontend Development (React, Next.js)
2. **backend-icon.png** - Icon untuk Backend Development (Node.js, APIs)
3. **mobile-icon.png** - Icon untuk Mobile Development (Flutter, React Native)
4. **uiux-icon.png** - Icon untuk UI/UX Design (Figma)

## Recommended Specs:
- Format: PNG dengan transparent background
- Size: 256x256px to 512x512px (square)
- File size: Under 150KB
- Style: Modern, colorful, professional

## Design Ideas:
- **Frontend**: Palette icon, browser window, React logo
- **Backend**: Server, database, gear icon
- **Mobile**: Phone/tablet, app screens
- **UI/UX**: Pen tool, design layers, Figma icon

- **Create Your Own**:
  - Canva (dengan templates)
  - Figma (design custom)
  - Adobe Illustrator

## Tips:
- Keep consistent style across all icons
- Use vibrant colors
- Ensure icons are clear at small sizes
- Transparent background is a must
- Can use 3D icons for modern look
